4分钟了解网页设计

原文： Jeremy Thomas

比如说，你有一个产品, 或是一个作品集, 或仅仅是一个想法，想在 你自己的 网页上分享给任何人。 在你发布到网上之前，你希望这些看上去要吸引人，有个性，至少是 得体 得体的。

那么什么事是你 第一步 要去做的呢？

内容

页面设计 的目标是提高内容呈现的吸引力。显然内容作为网页的 首要 元素应当是完整的，而不是当作后来才添加进来的东西。

文本内容，就像你正在读的这一段，组成了页面90%以上的部分。将文本内容添加上样式会对页面的呈现有很大的帮助。

假设你已经完成了你要呈现的内容，并且你刚新建了一个空白的style.css文件，那么什么才是你首先 要写的规则呢？

居中

连续文本过长是不易辨析的，以至不易阅读。为每行设置一个字符长度的限制，能极大提高一大块文本的可读性和吸引力。

body {
  margin: 0 auto;
  max-width: 50em;
}

为文本块添加了样式之后，那么关于文字自身的样式呢？

字体

浏览器默认的字体是"Times"（译注：中文是"宋体"），一种不太吸引人的字体 (主要因为是这种字体是“未样式化的”). 选择一种无衬 字体，如 "Helvetica" 或者 "Arial" 能极大地提高

body {
  font-family: "Helvetica", "Arial", sans-serif;
}

如果你想使用一种有衬字体，尝试 "Georgia"字体。

在让文字变得更具吸引力的同时，让我们使文字 更加可读。

间隙

在用户看了，一个页面显得很“破碎”，一般都是 间隙 的问题。为页面内容提供一个 围绕 且 不溢出 的间隙，会提高页面的吸引力。

body {
  line-height: 1.5;
  padding: 4em 1em;
}

h2 {
  margin-top: 1em;
  padding-top: 1em;
}

目前布局已经得到了极大的改善, 我们加上更多 细节调整。

颜色 和 反差

黑色文字在白色背景上是刺眼的. 为主体文本选择一种柔和的黑色使页面阅读显得更 舒服 。

body {
  color: #555;
}

为了同时保持得体的反差，我们为 关键词 选择一种深暗色

h1,
h2,
strong {
  color: #333;
}

当页面的大部分看上去得到改进, 一些元素（如代码片段）依旧显得 不合适。

匀称

仅需要几步就能修正页面的匀称:

code,
pre {
  background: #eee;
}

code {
  padding: 2px 4px;
  vertical-align: text-bottom;
}

pre {
  padding: 1em;
}

此刻，你或许想使你的页面更突出 个性。

基色

大多数商标都会有个基色 作为色调. 在一个网站中, 这种色调可以用于强调一些交互的元素, 如 超链接。

a {
  color: #e81c4f;
}

不过要保持匀称感，我们需要一些 别的颜色。

间色

色调可以辅以 精细 阴影, 可以使用在边框，背景，甚至是主体文本。

body {
  color: #566b78;
}

code,
pre {
  background: #f5f7f9;
  border-bottom: 1px solid #d8dee9;
  color: #a7adba;
}

pre {
  border-left: 2px solid #69c;
}

既然改变了阴影，为何不改变形状...

个性化字体

既然文本是整个网页的主要内容, 使用一种 个性化字体会给网页带来更多特色。

你可以嵌入你自己的网页字体，或使用一种在线服务如 Typekit,让我们使用 "Roboto" 字体，来自 Google Fonts 服务:（译注：大陆地区要体验该项服务是困难的。）

@import 'https://fonts.googleapis.com/css?family=Roboto:300,400,500';

body {
  font-family: "Roboto", "Helvetica", "Arial", sans-serif;
}

在提升你 文本的个性后, 不如再加上千言万语...

图片和图标能用于作为支持你内容的装饰, 或作为一个你想要传达的观点

让我们为页眉添加一个背景图片，从 Unsplash挑一张

header {
  background-color: #263d36;
  background-image: url("header.jpg");
  background-position: center top;
  background-repeat: no-repeat;
  background-size: cover;
  line-height: 1.2;
  padding: 10vw 2em;
  text-align: center;
}

我们同时添加一个 logo（标志）

header img {
  display: inline-block;
  height: 120px;
  vertical-align: top;
  width: 120px;
}

Let's take that opportunity to enhance the text styles.

header h1 {
  color: white;
  font-size: 2.5em;
  font-weight: 300;
}

header a {
  border: 1px solid #e81c4f;
  border-radius: 290486px;
  color: white;
  font-size: 0.6em;
  letter-spacing: 0.2em;
  padding: 1em 2em;
  text-transform: uppercase;
  text-decoration: none;
  transition: none 200ms ease-out;
  transition-property: color, background;
}

header a:hover {
  background:  #e81c4f;
  color: white;
}

And voilà!

We've designed a decent page in just a few minutes, following basic principles of web design. There's only one last thing left to do...
